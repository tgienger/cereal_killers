Website Settings

Default values for the website.

News Settings

Setting the news item to private will only allow users of the 'member' group or higher to view it. Leave unchecked to allow all visitors to view. (each post editable)

Setting the news item to visible will allow it to be seen. Leaving unchecked will prevent anyone besides the admin from viewing. (each post editable)

User Admin Settings

Checking this will require you to confirm changes made to users roles. (admin will always require confirmation)
